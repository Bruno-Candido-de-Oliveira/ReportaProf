# ReportaProf
Projeto ReportaProf (AMS - WEB II - Mobile)

Para inicializar o back-end, após realizar o download e extrair o arquivo em uma pasta, siga os passos abaixo:

1 - Criar a variável de ambiente com o comando no terminal "python -m venv venv"
2 - Inicializar a variável de ambiente "venv\Scripts\activate"
3 - Instalar as dependências "pip install -r requirements.txt"
4 - Rodar a aplicação "python manage.py runserver"
